# Amazon Delivery App UI Enhancement - Notify of Arrival Feature

![Project Banner](./assets/banner-amazon-delivery-enhancement.png)

A case study proposing a standalone "Notify of Arrival" bell-icon button for Amazon's delivery app to streamline notifications, reduce time by 80%, and save £156-167M annually across the UK.

## 🚀 Project Overview

![Before vs After Comparison](./assets/before-after-comparison.png)

This solution enhances driver efficiency and delivery success using **Design Thinking**, **Lean UX**, **Jobs to be Done (JTBD)**, **SWOT Analysis**, and **Agile Methodology**.

### Key Metrics Dashboard
![Key Metrics Infographic](./assets/key-metrics-dashboard.png)

- **Time Reduction**: 15s → 3s (-80%)
- **Driver Time Saved**: 50-58 hrs/driver/year
- **Cost Savings**: £6,250/driver, £156-167M UK-wide
- **Success Rate Lift**: 80% → 85% (+5 pp)

## 📋 Problem Statement

![Current Workflow Pain Points](./assets/current-workflow-problems.png)

Current process: 3-4 taps, 15s per notification (45s at 3-customer stops), leading to inefficiencies and missed deliveries.

### Design Principles
- **Fitts's Law**: Larger, accessible bell icon reduces tap time.
- **Proximity**: Placed beside info icon for intuitiveness.
